
# 📝 React Todo List

A modern and responsive Todo List application built with **React** and **Tailwind CSS**.

This project was created to practice React fundamentals including state management, props, component composition, and event handling while building a real-world application.

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/react-todo-list.git
```

Go to the project

```bash
cd react-todo-list
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm start
```
